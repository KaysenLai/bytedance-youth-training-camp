#### 开发思想

从问题出发：解决小问题，不断优化

#### Rollup

webpack  => esm => require(自定义，动态的)，静态的时候分析不出来

天然支持esm，esm => 静态 ast

能静态分析代码信息，生成依赖图

1. index.js 依赖于foo
2. foo.js 依赖于 {}

得到对应的代码信息 => 数据结构=>导出生成代码

没使用的代码不会打包进去
